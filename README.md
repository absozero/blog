## About The Project


This is a blog, hosted on github pages, that serves to be the blog of the user Absozero. It is their location for posting information and it uses jekyll. It is a blog that strives to be different from every other one, in both content and style.
The site can be accessed at [blog](https://absozero.github.io/blog).
Due to being a blog and the nature of a blog, this site will not need much structural support, but it will strive to be updated whenever Absozero gets time to update it.

## Getting Started

To use jekyll locally to build the blog, you can use the following steps. Or, you could fork this repo and setup a github pages in another repo, to serve the content.

### Dependencies

To use jekyll, you need to install the jekyll and bundler gems from ruby. This requires ruby to be installed(*duh*), and you can follow [these](https://www.ruby-lang.org/en/documentation/installation/) resources to do so.
Then, you can install jekyll and bundler with gem
```bash
gem install bundler jekyll
```
Then clone the repo with git or with gh
```bash
gh repo clone absozero/blog
or
git clone https://github.com/absozero/blog.git
```
After, run `cd blog` and run `jekyll serve` to run a development server with jekyll.

### Installation

*Once dependencies are set, use these commands to set up this blog locally.*

1. Clone the repo with git or with gh
```bash
gh repo clone absozero/blog
or
git clone https://github.com/absozero/blog.git
```
2. Run `cd blog` 
3. Run `jekyll serve` to run a development server with jekyll on port 4000.
